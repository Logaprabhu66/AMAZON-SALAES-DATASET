AMAZON SALES DATA ANALYSIS
1. Introduction

The rapid growth of e-commerce platforms has generated massive amounts of sales data. This project focuses on analyzing an Amazon sales dataset to derive meaningful insights using data visualization techniques. The dashboard provides a clear understanding of sales performance, customer behavior, and product trends.

2. Objective of the Project

To analyze Amazon sales data efficiently

To identify top-performing product categories

To understand customer purchasing behavior

To evaluate revenue generation across regions

To build an interactive dashboard for decision-making

3. Dataset Description

The dataset used in this project contains the following key fields:

Product Category – Type of product (Beauty, Books, Electronics, etc.)

Quantity Sold – Number of items sold

Rating – Customer ratings

Review Count – Number of reviews per product

Revenue – Total sales revenue

Customer Region – Asia, Europe, Middle East, North America

Payment Method – COD, Credit Card, Debit Card, UPI, Wallet

Year – Sales data categorized by year (2022, 2023)

4. Tools & Technologies Used

Power BI / Tableau – Data visualization

Excel / CSV – Data source

Data Cleaning Techniques – Handling missing values and formatting

7. Key Insights

📈 Sales increased from 2022 to 2023

🏆 Sports category is the top-performing segment

🌍 North America contributes the highest sales

💳 Digital payments (UPI, Cards) dominate transactions

⭐ High review count indicates strong customer engagement

8. Advantages of the Dashboard

Easy to understand visual representation

Identifies business growth opportunities

Tracks customer behavior effectively

9. Limitations

No real-time data updates

External factors (seasonality, marketing) not included
